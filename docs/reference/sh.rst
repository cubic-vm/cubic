.. _ref_cubic_sh:

cubic sh
========

.. code-block::

    $ cubic sh --help
    Open a shell in a virtual machine instance

    Usage: cubic sh [OPTIONS] <INSTANCE>

    Arguments:
      <INSTANCE>  Name of the virtual machine instance

    Options:
      -v, --verbose  Enable verbose logging
      -q, --quiet    Reduce logging output
      -h, --help     Print help
