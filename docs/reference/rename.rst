.. _ref_cubic_rename:

cubic rename
============

.. code-block::

    $ cubic rename --help
    Rename a virtual machine instance

    Usage: cubic rename <OLD_NAME> <NEW_NAME>

    Arguments:
      <OLD_NAME>  Name of the virtual machine instance to rename
      <NEW_NAME>  New name of the virtual machine instance

    Options:
      -h, --help  Print help
