.. _ref_cubic_show:

cubic show
==========

.. code-block::

    $ cubic show --help
    Show a virtual machine instance

    Usage: cubic show [OPTIONS] <INSTANCE>

    Arguments:
      <INSTANCE>  Name of the virtual machine instance

    Options:
      -v, --verbose  Increase logging output
      -q, --quiet    Reduce logging output
      -h, --help     Print help
