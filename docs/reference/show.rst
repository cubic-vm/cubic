.. _ref_cubic_show:

cubic show
==========

.. code-block::

    $ cubic show --help
    Show virtual machine image or instance information

    Usage: cubic show [OPTIONS] <NAME>

    Arguments:
      <NAME>  Name of the virtual machine image or instance

    Options:
      -v, --verbose  Increase logging output
      -q, --quiet    Reduce logging output
      -h, --help     Print help
