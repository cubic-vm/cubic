cubic mount
===========

.. code-block::

    $ cubic mount --help
    Mount subcommands

    Usage: cubic mount <COMMAND>

    Commands:
      list  List mount mounts
      add   Add a directory mount
      del   Delete a directory mount
      help  Print this message or the help of the given subcommand(s)

    Options:
      -h, --help  Print help
