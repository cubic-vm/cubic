.. _ref_cubic_console:

cubic console
=============

.. code-block::

    $ cubic console --help
    Open the console of an virtual machine instance

    Usage: cubic console [OPTIONS] <INSTANCE>

    Arguments:
      <INSTANCE>  Name of the virtual machine instance

    Options:
      -v, --verbose  Increase logging output
      -q, --quiet    Reduce logging output
      -h, --help     Print help
