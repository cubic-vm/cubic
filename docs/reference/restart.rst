.. _ref_cubic_restart:

cubic restart
=====

.. code-block::

    $ cubic restart --help
    Restart virtual machine instances
    
    Usage: cubic restart [OPTIONS] [INSTANCES]...
    
    Arguments:
      [INSTANCES]...  Name of the virtual machine instances to restart
    
    Options:
      -v, --verbose  Increase logging output
      -q, --quiet    Reduce logging output
      -h, --help     Print help
